# Installation

Add the package to your Flutter app:

```yaml
dependencies:
  animated_streaming_markdown: ^0.3.2
```

Install dependencies:

```sh
flutter pub get
```

Import the public API:

```dart
import 'package:animated_streaming_markdown/animated_streaming_markdown.dart';
```

## Requirements

- Flutter `>=3.0.0`
- Dart SDK `>=2.17.0 <4.0.0`
- Native build toolchain for your target platform

Because the package includes native parser integration, app targets need the
usual platform build setup for Android, iOS, macOS, Linux, or Windows.
